Calculation Methodology

Source

All lookup tables, prices, and formulas are derived from the official MuleSoft UBP sizing workbook. Version shown in the page header.

Simplified — Flows (T-shirt sizing)

Flows are determined by a two-axis matrix: Systems × Business Objects → T-shirt size (S/M/L/XL/XXL) → flow count by integration type. Flows do not scale with environments.

Simplified — Messages / Month

messages = tasks_range × bidir_multiplier × (1 − reusability) × num_environments

Simplified — Data Throughput / Month

data_GB = messages × payload_KB × direction_factor / 1,048,576

Direction factor: 4 for bidirectional, 2 for unidirectional.

Detailed — Flows per Component

Flows are calculated precisely per component based on integration type:

TypeFormulaDefaults
API / Microservicesoperations + API Console + HTTP Listener4 + 1 + 1 = 6
Event-Basedqueues / channels / event types1
Schedule-Basedentities × (bidirectional ? 2 : 1)1 × 1 = 1

HA components are multiplied by the HA replica count. Flows then scale by production and non-production environment counts, plus the flows buffer.

Detailed — Messages (annual)

annual_msgs = tasks × time_multiplier(period, bizHours, bizDays)

Time multipliers: Second = 3,600 × hours × days  |  Minute = 60 × hours × days  |  Hour = hours × days  |  Day = days  |  Month = 12  |  Year = 1.

Detailed — Environment Scaling

prod_msgs = base_msgs × prod_envs × (1 + msg_buffer)
np_msgs = base_msgs × non_prod_envs × non_prod_pct × (1 + msg_buffer)
prod_flows = base_flows × prod_envs × (1 + flows_buffer)

Detailed — API Management Totals

Counted only for MuleSoft components with type API / Microservices.

APIs to Manage (Prod) = # API/MS components × prod_envs
APIs to Manage (Non-Prod) = # API/MS components × non_prod_envs
APIs to Govern = APIs to Manage (Prod only)

Detailed — Omni Gateway

Components implemented as "Omni Gateway" represent APIs hosted outside Anypoint Platform. Their task volume counts as Omni Gateway API calls, sized separately from MuleSoft runtime messages.

Premium Connector Licensing

Pricing

Each premium connector is licensed at $33,600 / environment / year (list price). Production and non-production environments are billed at the same rate.

Environment Scope

A connector must be licensed for each deployment environment where it is used — Production, Pre-Production, Staging, Dev, DR, etc. Count every environment separately.
